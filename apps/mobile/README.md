# CluePot Mobile

React Native (Expo) 기반 모바일 앱입니다.

프로젝트 전체 소개는 [루트 README](../../README.md)를 참고해주세요.

## 실행 방법

1. 의존성 설치

   ```bash
   cd apps/mobile
   npm install
   ```

2. 개발 서버 실행

   ```bash
   npx expo start
   ```

## 환경변수

`apps/mobile/.env` 파일을 생성하고 아래 값을 채워주세요.

```env
EXPO_PUBLIC_API_URL=
EXPO_PUBLIC_SUPABASE_URL=
EXPO_PUBLIC_SUPABASE_PUBLISHABLE_KEY=
EXPO_PUBLIC_KAKAO_MAP_KEY=
EXPO_PUBLIC_WEB_URL=
GEMINI_API_KEY=
```
